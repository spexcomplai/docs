Once all sections have been reviewed and compliance statuses are confirmed, you can export the results.

## Download the report

Click the **Download Excel** icon from the RFP Viewer toolbar.

<Frame>
  <img src="/images/how-to-guide/download-excel-button.png" alt="Download Excel button highlighted in the RFP Viewer toolbar" />
</Frame>

## Report contents

The exported Excel file includes:

| Column | Description |
| --- | --- |
| Requirement reference | Original numbering from the RFP |
| Compliance status | Applied status (Compliant, Partially Compliant, Non-Compliant) |
| Confidence score | System confidence in the evaluation |
| Reasoning | AI-generated explanation |
| Source references | Page numbers from the linked datasheet |
| SKU details | Per-SKU evaluation results |

<Frame>
  <img src="/images/how-to-guide/excel-export-example.png" alt="Excel export showing columns for section, requirement text, status, confidence, reasoning, and references" />
</Frame>

The structure of the Excel file mirrors the RFP hierarchy to maintain traceability and alignment with the original document.

<Info>
The Excel export format is still evolving as part of the Early Access phase. All compliance data is fully available and structured. The file is ready for customization, filtering, and formatting as needed for submission or internal review.
</Info>

## Workflow complete

You have now completed the full compliance workflow:

**Create** → **Review** → **Configure** → **Run** → **Validate** → **Export**

Following this sequence ensures accurate, structured, and traceable compliance evaluation.
